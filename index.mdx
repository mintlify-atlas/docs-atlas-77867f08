luci-app-mosdns brings [MosDNS v5](https://github.com/IrineSistiana/mosdns) to OpenWrt with a full LuCI web interface. It routes DNS queries intelligently — China-region domains go to local resolvers, international domains go to encrypted upstream servers — while supporting every modern DNS protocol and giving you fine-grained control over caching, filtering, and custom rules.

## How it works

luci-app-mosdns configures MosDNS as a plugin-based DNS forwarder running on your router. Incoming DNS queries pass through a sequenced pipeline of plugins that match domain names and IP addresses against geo databases, apply your custom rules, check the cache, and forward to the appropriate upstream server.

<Steps>
  <Step title="Install on your router">
    SSH into your OpenWrt device and run the one-line install script to download and install all required packages for your CPU architecture.
  </Step>
  <Step title="Enable and configure in LuCI">
    Navigate to **Services → MosDNS** in the LuCI web interface. Enable the service, set your listen port, and choose your upstream DNS servers.
  </Step>
  <Step title="Set up rules and filtering">
    Use the **Rules** tab to add domains to whitelist, blocklist, greylist, or DDNS lists. Enable ad blocking and configure geo-routing behavior.
  </Step>
  <Step title="Update geo databases">
    Visit the **Geodata Update** tab to configure automatic scheduled updates for GeoIP and GeoSite databases, keeping your routing rules current.
  </Step>
</Steps>

## Supported DNS protocols

| Protocol | Example format |
|----------|---------------|
| UDP | `119.29.29.29` or `udp://119.29.29.29:53` |
| TCP | `tcp://119.29.29.29` or `tcp://119.29.29.29:53` |
| DNS-over-TLS | `tls://120.53.53.53` or `tls://120.53.53.53:853` |
| DNS-over-HTTPS | `https://120.53.53.53/dns-query` |
| DNS-over-HTTPS (HTTP/3) | `h3://dns.alidns.com/dns-query` |
| DNS-over-QUIC | `quic://dns.alidns.com` or `doq://dns.alidns.com` |

<Note>
  luci-app-mosdns v5 requires OpenWrt 21.02 or later with LuCI installed. The prebuilt install script supports both `opkg` (21.02–24.10) and `apk` (25.12+) package managers.
</Note>
